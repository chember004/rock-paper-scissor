# Rock-Paper-Scissor Laravel11 & Nextjs15

Starter

## Local Installation

Install Docker Windows.

```
docker-compose up --build -d
```

Enjoy

## Docker Compose

Run service

```
docker-compose up -d
```

Stop service

```
docker-compose down
```

Open Frontend

# http://localhost:3000/

for Backend

# http://localhost:8000/
